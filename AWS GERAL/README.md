# AWS GERAL

Amazon Web Services

O Amazon Web Services (AWS) é uma plataforma de computação em nuvem da Amazon que oferece diversos serviços para armazenamento, processamento, bancos de dados, redes e muito mais.

Principais Vantagens
Escalabilidade: aumente ou diminua recursos conforme a demanda
Alta disponibilidade: infraestrutura distribuída globalmente
Pagamento sob demanda: pague apenas pelo que usar
Segurança: diversos níveis de proteção e certificações
Flexibilidade: suporte para diferentes linguagens e frameworks

Serviços Mais Utilizados

EC2 (Elastic Compute Cloud): criação de máquinas virtuais
S3 (Simple Storage Service): armazenamento de arquivos na nuvem
RDS (Relational Database Service): bancos de dados gerenciados
Lambda: execução de código sem servidor (serverless)
CloudFront: distribuição de conteúdo (CDN)IAM (Identity and Access Management): controle de acesso e permissões

Regiões e Zonas de Disponibilidade

A AWS possui data centers distribuídos pelo mundo em regiões e zonas de disponibilidade (AZs), garantindo maior desempenho e redundância.

Modelo de Preços

Free Tier: camada gratuita com limites mensais
Pay-as-you-go: cobrança conforme uso
Reservas: descontos para uso contínuo

Segurança

* Criptografia de dados em trânsito e em repouso
* Controle de acesso com IAM
* Monitoramento com CloudWatch e CloudTrail

Casos de Uso

* Hospedagem de sites e APIs
* Armazenamento de arquivos
* Machine Learning e IA
* Big Data e análise de dados
* Aplicações serverless
