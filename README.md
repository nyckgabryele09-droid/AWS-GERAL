# AWS GERAL
O que é a AWS

A Amazon Web Services (AWS) é uma plataforma de computação em nuvem criada pela Amazon que oferece diversos serviços como armazenamento, processamento, bancos de dados, redes e inteligência artificial.

Ela permite que empresas e desenvolvedores utilizem recursos de tecnologia sem precisar investir em infraestrutura física, acessando tudo pela internet.

Infraestrutura Global

A AWS possui uma infraestrutura global altamente distribuída, composta por:

Regiões (Regions)

São áreas geográficas ao redor do mundo onde os serviços da AWS estão disponíveis.
Cada região é independente e localizada em diferentes países ou continentes.

Zonas de Disponibilidade (Availability Zones)

Dentro de cada região existem várias zonas de disponibilidade.
Cada zona é formada por um ou mais data centers físicos, isolados entre si, garantindo:

Alta disponibilidade
Tolerância a falhas
Melhor desempenho

Essa estrutura permite que aplicações continuem funcionando mesmo se uma zona apresentar problemas.

Economia da Nuvem
Modelo Pay-as-you-go

A AWS utiliza o modelo “pague pelo uso”, ou seja:

Você paga apenas pelos recursos que utiliza
Não há necessidade de investimento inicial alto
É possível escalar os serviços conforme a necessidade
Economia de Escala

A AWS consegue reduzir custos devido à sua grande escala de operação:

Compra de hardware em grande quantidade
Operação global eficiente
Compartilhamento de infraestrutura entre milhões de usuários

Isso permite oferecer preços mais baixos enquanto ainda gera lucro, já que o volume de clientes é muito alto.
